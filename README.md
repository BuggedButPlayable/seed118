# S.E.E.D 118

S.E.E.D 118 is an open-source factory and automation game playable in the browser.

The repository contains both the game itself and the project homepage. Development happens in public and focuses on clarity, systems design, and long-term iteration.

---

## Repository Structure

- `game/` — the game client and simulation
- `homepage/` — the project website
- `.github/` — contribution guidelines and meta files

---

## Inspiration

This project is conceptually inspired by shapez and similar factory games. The inspiration is philosophical rather than literal.
